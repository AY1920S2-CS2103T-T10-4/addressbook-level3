== PROJECT: Nice and Amazing Student Assistant (NASA)

---

== Overview

Nice & Amazing Student Assistant (NASA) is an application for busy university students to manage their module
assignments, lessons, exams and consultations etc.

== Summary of contributions

* *Major enhancement*: added *ability to re-schedule an activity for a period of time (weekly, twice a week, monthly).*
** What is does: allows user to repeat a certain activity for as long as the user wants.
** Justification: save the amount of time user spent on doing repetitive tasks.
** Future enhancement: allow more flexibility for users to specify specific days to repeat instead of current presets
of (weekly, twice weekly and monthly).

* *Major enhancement*: added *the ability to undo/redo previous commands*
** What it does: allow user to restore the previous state of Nasa and previous appearance of UI.
** Justification: This feature allows user to rectify the error made.
** Highlights: Current implementation fits the following functionality of Nasa, however when more functions is added in v2.0, update will be needed on the code for tracking the UI looks.

* *Minor enhancement*: added a history command that allows the user to navigate to previous commands using up/down keys.

* *Minor enhancement*: added a quote command that aims to motivate the user before he/she starts to get into work.
